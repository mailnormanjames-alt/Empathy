# EMPATHY - India's No. 1 Web Design Agency

A high-performance, human-centered agency website built with **Next.js 14**, **Contentlayer (MDX)**, and **Framer Motion**.

## 🚀 Quick Start

1. **Install**: `npm install`
2. **Build**: `npm run build`
3. **Dev**: `npm run dev`

## 📦 Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Content**: Contentlayer + MDX
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion

## 🛠 Deployment (Vercel)

1. Push your code to a GitHub repository.
2. Import the repo at [Vercel](https://vercel.com/new).
3. Done! Vercel handles the build and deployment.
