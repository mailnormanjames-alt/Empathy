# EMPATHY WEB DESIGN AGENCY - INDIA'S NO. 1 WEB DESIGN AGENCY

This is a modern, high-performance website for Empathy Web Design Agency, built with Next.js 14+, MDX, Contentlayer, Tailwind CSS, and Framer Motion.

## Features

- **Next.js 14+ (App Router)**: Utilizing the latest Next.js features for optimal performance and SEO.
- **MDX with Contentlayer**: Type-safe content handling for blog posts, case studies, and services.
- **Tailwind CSS**: Custom theme with a warm, approachable color palette inspired by human-centered design.
- **Framer Motion**: Subtle, natural animations that respond with personality and warmth.
- **Responsive Design**: Works beautifully on all devices, from mobile to desktop.
- **SEO & Schema Markup**: Optimized for the Indian market with dynamic metadata and local business schema.
- **Human-Centered Design**: Every UI decision reflects the core philosophy of empathy and understanding.

## Tech Stack

- **Framework**: [Next.js 14+](https://nextjs.org/)
- **Content**: [Contentlayer](https://www.contentlayer.dev/) with [MDX](https://mdxjs.com/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Forms**: [React Hook Form](https://react-hook-form.com/) with [Zod](https://zod.dev/)
- **Deployment**: [Vercel](https://vercel.com/)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/empathy-agency.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app`: App Router pages and layouts.
- `src/components`: Reusable UI components.
- `src/lib`: Utility functions and helpers.
- `src/styles`: Global styles and Tailwind configurations.
- `content`: MDX content for blog, projects, and services.

## Deployment

This project is ready to be deployed on Vercel. Simply push your code to a GitHub repository and connect it to Vercel for automatic deployments.

### Environment Variables

If you're using Resend for email integration, add the following environment variable to your Vercel project:

- `RESEND_API_KEY`: Your Resend API key.
